# Deep Learning Speech Classification Project

A deep learning-based speech and audio classification project implemented in MATLAB.

This project focuses on speech signal processing, spectrogram analysis, and convolutional neural network (CNN) based classification methods.

---

# Project Overview

The objective of this project is to explore how deep learning techniques can be applied to speech and audio recognition tasks.

The project includes:

* Audio preprocessing
* Spectrogram generation
* CNN-based classification
* Training and validation pipelines
* Performance evaluation

---

# Features

## Speech Signal Processing

Audio signals are processed and converted into spectrogram representations suitable for deep learning models.

## CNN Classification

Convolutional Neural Networks (CNNs) are used to classify speech/audio data.

## Training and Validation

Separate training and validation datasets are used to evaluate model performance.

## Performance Evaluation

The project includes:

* Accuracy analysis
* Confusion matrix evaluation
* Classification performance comparison
